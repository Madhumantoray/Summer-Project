StockResearch TODO

Quick start, pipeline steps, data flow, and research end goal.

Start Here

  • Backend: open PowerShell in D:\StockResearch\backend, create/activate venv, install requirements, then run uvicorn main:app --reload.
  • Frontend: open PowerShell in D:\StockResearch\frontend and run npm run dev.
  • Open the UI at http://localhost:3000.

Scripts To Run (What They Do)

  • scripts/run_daily_pipeline.py — orchestrates full daily pipeline (news, sentiment, prices, indicators, research metrics).
  • scripts/collect_news.py — fetches Google News RSS and scores headlines with FinBERT.
  • scripts/collect_stock_data.py — downloads OHLCV + computes RSI/MACD.
  • scripts/build_research_dataset.py — builds forward returns + abnormal returns dataset.

How The DB Forms

  • SQLite DB file auto-creates at backend\database\stocks.db when the FastAPI app starts.
  • Tables are created on startup via the FastAPI lifespan hook.
  • Primary tables: news_sentiment, stock_prices, research_metrics.

Data Flow (Short)

News RSS + yfinance fallback -> Headline cleaning -> FinBERT sentiment scoring -> news_sentiment table Daily stock prices (OHLCV) -> RSI/MACD indicators -> stock_prices table news_sentiment + stock_prices -> forward returns + abnormal returns -> research_metrics table

Pipeline / Workflow (Short)

  • Run the pipeline daily after market close (16:30 IST recommended).
  • Each step is idempotent: safe to re-run without duplicate data.
  • Use the research endpoints to validate data volume and correlations.

End Goal Of The Research

  • Quantify whether news sentiment predicts short-horizon returns for NSE equities.
  • Measure statistical strength with correlation and event study methods.
  • Build a reliable research dataset that grows with daily automation.

8. Hypothesis Testing

  • Hypothesis: negative news often makes a stock go up, and positive news often makes a stock underperform.
  • Related to: contrarian investing, market overreaction, behavioral finance.
  • Possible reasons: bad news already priced in, panic selling creates undervaluation, positive expectations become too optimistic.

9. How To Scientifically Test It

  • Need sentiment score at time T and future returns after T.
  • Example: sentiment on Day D, then measure returns after 1, 3, 7, and 30 days.
  • Compute correlation: corr(sentiment, future_return) and regression: future_return ~ sentiment.
  • Event study: group strongly negative vs strongly positive news and compare average future returns.

10. Expected Reality (End Goal)

  • Short-term: negative news leads to a temporary drop, then mean reversion.
  • Long-term: fundamentals dominate sentiment.
  • Meme or high-volatility stocks: sentiment impact is much stronger.

How To Achieve It Without Issues

  • Run the backend once to create tables before running pipeline scripts.
  • Keep a stable internet connection for RSS, yfinance, and FinBERT downloads.
  • Schedule the daily pipeline after market close to avoid partial day data.
  • Monitor logs for errors and re-run the pipeline if a step fails.
  • Delete stocks.db only if you want a clean rebuild.
